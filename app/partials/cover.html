AngularJS

AngularJS est un framework open-source écrit dans le langage javascript. Ce framework a été développé par Google et s’appuie sur le retour d’expérience des équipes qui ont développé les applications Google tel que Gmail, Maps et autres.

En savoir plus

Node.js

Node.js permet d'utiliser JavaScript côté serveur, en dehors du navigateur. Avant de pouvoir exécuter votre JavaScript sur le serveur, le code a besoin d'être interprété. C'est précisément ce que fait Node.js en utilisant le moteur V8 de Google.

En savoir plus

Bootstrap

Twitter Bootstrap est une collection d'outils utile à la création de sites et applications web. C'est un ensemble qui contient des codes HTML, CSS et Javascript pour des formulaires, boutons, outils de navigation et autres éléments interactifs.
