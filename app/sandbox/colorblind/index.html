Test for colorblind

Test for issue https://github.com/twbs/bootstrap/issues/14744.

Uncheck for retrieving the original bootstrap color scheme.

Note: The color has been specified with RGB format for browser compatibility. Regarding the border color effect, for a given class (success, warning, error), the HUE and the SATURATION are the same and only the LUMINOSITY is increased to give a perception of "being under the spot".
The green color SATURATION is high but not the maximum to avoid the feeling of "couleur pétante" (flashy color).

Test for colorblind with background

Test for issue https://github.com/twbs/bootstrap/issues/14744.

Uncheck for retrieving the original bootstrap color scheme.

Note: This solution W3C WCAG AAA compliant according the color contrast ratio with http://www.snook.ca/technical/colour_contrast/colour.html .
However, the indication of the status is perceived by the pastel background color (same hue with high luminosity and low saturation).
